# gtk_grid
Making grid of buttons uing only GTK (no extra files like grid.ui).

Using unsafe block in quit_button closure.

grid.attach(&button, 0, 0, 1, 1);

1 - Position from left to the right

2 - Position from top to the bottom

3 - Width

4 - Height
